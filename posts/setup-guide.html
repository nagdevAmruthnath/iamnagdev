Building your blog with the right structure

April 19, 2026Setup3 min read

This document outlines the architecture and requirements for the iamnagdev blog, built for GitHub Pages with a clean, modern layout inspired by a professional blog theme.

Use a clear repository structure, strong navigation, and simple post templates to ensure every article loads fast, looks polished, and is easy to maintain.

Essential Requirements

  • Create a public repository named iamnagdev.github.io.
  • Use HTML, CSS, and JavaScript for page structure, style, and interactions.
  • Write article pages under posts/ and link them from the homepage.
